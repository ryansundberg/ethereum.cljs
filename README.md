# ethereum.cljs
ClojureScript namespaces for ethereum.js

# Usage

``[ethereum.cljs "0.3.0-SNAPSHOT"]``

    (:require
      [eth.js.web3 :as web3]
      [eth.js.eth :as eth]
      [eth.js.shh :as shh])

# Running tests
    cd ethereum.cljs
    lein run
    # firefox http://localhost:8081
